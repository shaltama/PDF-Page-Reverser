# PDF页面颠倒工具使用说明

## 功能介绍
这个GUI程序可以完全颠倒PDF文件的页面排序，将第一页变为最后一页，第二页变为倒数第二页，以此类推。所有PDF信息都会完整保留，包括文本、图片、格式等。

## 主要特点
- 🎯 **完全颠倒页面顺序**：第一页变成最后一页，保持完美对称
- 🔄 **信息完整保留**：所有PDF内容、格式、元数据都不会丢失
- 🖥️ **图形化界面**：简单易用的GUI，无需命令行操作
- 📊 **进度显示**：实时显示处理进度和状态
- 🛡️ **错误处理**：完善的错误提示和异常处理
- 📁 **智能命名**：自动生成输出文件名，避免覆盖原文件


## 使用方法

### 1. 启动程序
双击运行 `PDF页面颠倒工具.exe` 文件

### 2. 选择输入文件
- 点击"浏览..."按钮
- 选择要处理的PDF文件
- 程序会自动生成输出文件名

### 3. 选择输出文件（可选）
- 如果需要，可以修改输出文件的位置和名称
- 默认会在原文件同目录下生成 `原文件名_reversed.pdf`

### 4. 颠倒页面
- 点击"颠倒页面顺序"按钮
- 等待处理完成
- 查看状态栏了解处理进度

### 5. 完成
- 处理完成后会显示成功消息
- 输出文件已生成，页面顺序已完全颠倒

## 示例

### 输入PDF（5页）：
```
第1页 → 第2页 → 第3页 → 第4页 → 第5页
```

### 输出PDF（5页）：
```
第5页 → 第4页 → 第3页 → 第2页 → 第1页
```

## 注意事项

1. **文件权限**：确保对输入文件有读取权限，对输出目录有写入权限
2. **文件大小**：大文件可能需要较长的处理时间
3. **内存使用**：处理大文件时会占用较多内存
4. **备份建议**：建议先备份重要的PDF文件

## 常见问题

### Q: 程序支持哪些PDF格式？
A: 支持所有标准的PDF文件，包括包含图片、表格、复杂格式的PDF。

### Q: 会丢失PDF的书签或链接吗？
A: 不会，所有PDF内容都会完整保留，包括书签、链接、注释等。

### Q: 可以处理加密的PDF吗？
A: 如果PDF有打开密码，需要先解密才能处理。程序目前不支持处理加密PDF。

### Q: 处理速度如何？
A: 处理速度取决于PDF文件大小和页数。一般几百页的PDF几秒钟内可以完成。
